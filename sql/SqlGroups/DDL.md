## DDL

`CREATE` - создание таблицы

```sql
CREATE TABLE "user" (name text, age  integer);
```

`ALTER` - изменение таблицы

```sql
ALTER TABLE "user" ADD COLUMN surname text;
```

`DROP` - удаление таблицы
```sql
DROP TABLE "user"
```

`TRUNCATE` - очистка таблицы
```sql
TRUNCATE TABLE "user";
```
