# Полезные приёмы для работы с консолью
- ```column -t```  - Выводит данные в видет таблицы

    ```-s``` - позволяет добавить разделитель (по умолчанию разделитель - пробелы)
    
    Пример c разделителем двоеточие: ``` mount | column -t -s : ```
    
---

- ```sort``` - Сортировка

    Сортировка процессов по памяти: ``` ps aux | sort -nk 4 ```
    
    Сортировка процессов по использованию ресурсов CPU: ``` ps aux | sort -nk 3 ```
    
---

- ```multitail``` - Просмотрол логов, как ```tail```, 
    но с возможностью простматривать несколько файлов одновременно
    
---

- ```nohup``` - Позволяет продолжить выполнение запущенной команды, даже если закроем терминал.
    
    Пример: ``` nohup bin/console app:import:products -e dev```
    
    Если закроем терминал, то выполнение команды продолжится. 
    
    Весь вывод можно посмотреть потом в файле ```nohup.out```
    
--- 

- ```Yes``` - Если при выполнении команды хотим, чтобы всегда подтверждались действия.

    Пример: ``` yes | apt-get update ``` 
    
    Для отрицательных ответов: ``` yes no | command ```
    
---

- ```!!``` - Выполненить последнюю команду

---

- ```script``` - Запись всего, что происходит в терминале

1. Вводим команду ```script```
2. Do something
3. Выходим из сессии ```exit```
4. Всё, что выводилось, попало в файл ```typescript```
